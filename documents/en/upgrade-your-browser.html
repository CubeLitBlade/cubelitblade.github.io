Microsoft has discontinued support for Internet Explorer

On June 15, 2022, Microsoft discontinued support for all versions of Internet Explorer, which means that unless you want to access some overly old, IE-only web pages, Internet Explorer is not recommended under any circumstances. Internet Explorer does not support many modern Web standards and has lost technical support, which may cause the page you request to display abnormally. Change your browser to display your requested page properly.

Why am I seeing this page?

You are attempting to access the page using Internet Explorer or the Internet Explorer mode of some browsers. However, Internet Explorer is not supported by the page you are requesting, so you are being told this information.

What should I do?

  • If you are using Internet Explorer, please change your browser;
  • If not, check if your browser has "Internet Explorer Mode" or "Compatibility Mode" enabled. If so, disable it and reload the page you requested.
