Microsoft 已停止对 Internet Explorer 的支持

早在2022年6月15日， Microsoft 就已停止对所有版本的 Internet Explorer 的支持。这意味着，在此之后，除非您需要访问一些过分老旧的页面，否则，在任何情况下都不推荐使用 Internet Explorer 了。

我们不得不承认，Internet Explorer 带领我们见证了互联网从起步到繁荣的将近三十个年头，也一度凭借其独特的优势占据了大量的份额。但时过境迁，如今的 Internet Explorer 在支持现代 Web 标准和技术方面远远落后于其他现代浏览器，这导致开发者需要花费更多时间和精力来解决兼容性问题，影响开发效率和用户体验。而且， Internet Explorer 在安全性方面存在多个漏洞和问题，这些问题可能被恶意攻击者利用来进行网络攻击。此外，相比现代浏览器，Internet Explorer 在性能方面表现不佳，包括加载速度、渲染速度和 JavaScript 执行速度等方面。从当下的种种情形来看，该向这个老家伙告别了。

我为什么会看到此页面？

我们注意到您正在使用 Internet Explorer，或者部分现代浏览器的“Internet Explorer 模式”（也被称为“兼容模式”）访问此页面。我们的网页已经不再支持 Internet Explorer，因此我们将这些内容告知于您。

我该怎么做？

如果您正在使用 Internet Explorer，请更换您的浏览器。如果您并没有使用 Internet Explorer 却看到了这个页面，请尝试关闭浏览器的”Internet Explorer 模式”或者“兼容模式”后，重新加载您所请求的页面。
